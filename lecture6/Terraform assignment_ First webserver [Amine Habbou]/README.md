# Terraform First Webserver - Amine Habbou

This is my first webserver deployed using Terraform and Docker.

## Files included:
- `main.tf` - Terraform configuration file
- `index.html` - The webpage HTML file
- `screenshot_page.png` - Browser showing the live webpage
- `screenshot_output.png` - Terraform output command result

## How to run:
1. Initialize: `terraform init`
2. Deploy: `terraform apply`
3. Access: http://localhost:8081
4. Destroy: `terraform destroy`
